What does an entity-relationship diagram represent?

Entity-relationship diagram represents a logical structure of the database graphically.

How is entity relationship set represented?

Diamond.

How are roles represented in E-R Diagram?

Roles are represented by labeling the lines that connect to a diamond with a rectangle.

What is Maximum cardinality?

Is the maximum number of entities that can be involved in a relationship.

What is a recursive relationship?

A recursive relationship is a relationship between an entity and itself.

What is an One-to-many relationship?

One-to-Many Relationship is a single-entity instance of one type related to many entity instances of another type.

What is a supertype entity?

A supertype entity is a type of entity that is related to two or more associated entities, that each contain specialized attributes that apply to some but not all of the instances of the entity.

How is a weak entity represented?

Weak entity set is represented by Double diamond.

What types of attribute are there?

Simple attribute, Composite attribute, derived attribute, Single-valued attribute, Multi-value attribute.
